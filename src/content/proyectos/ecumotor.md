## El reto

EcuMotor necesitaba una plataforma web que permitiera mostrar un sistema orientado a la **gestión y control del mantenimiento de vehículos**.

El objetivo del proyecto era desarrollar una web capaz de:

- Presentar una plataforma para **administrar información de vehículos**
- Permitir a los usuarios llevar control sobre **revisiones y mantenimientos**
- Mostrar alertas o recordatorios relacionados con servicios automotrices
- Transmitir una imagen moderna y tecnológica

Las plataformas de gestión vehicular permiten centralizar información importante del automóvil, como mantenimientos, revisiones y notificaciones preventivas para asegurar su correcto funcionamiento.

---

## La solución

Se desarrolló una **landing page moderna enfocada en soluciones digitales para el sector automotriz**, diseñada para comunicar claramente el funcionamiento de la plataforma.

La solución incluyó:

- Diseño UI/UX orientado a **herramientas de gestión automotriz**
- Secciones para explicar funcionalidades del sistema
- Presentación visual de alertas y recordatorios de mantenimiento
- Diseño responsive adaptado a dispositivos móviles
- Implementación con despliegue rápido en **Vercel**

El sitio permite que los usuarios comprendan cómo la plataforma puede ayudar a gestionar el estado de sus vehículos de manera simple.

---

## El resultado

Tras el lanzamiento del sitio web:

- **Presencia digital clara para la plataforma**
- Experiencia de usuario moderna y fácil de navegar
- Mejor comunicación del propósito del sistema
- Base preparada para futuras integraciones o funcionalidades

El proyecto permitió presentar EcuMotor como una solución digital para el seguimiento del mantenimiento vehicular y la gestión eficiente de información del automóvil.
